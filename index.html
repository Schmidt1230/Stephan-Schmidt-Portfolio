ChatterBox

ChatterBox is a Python desktop application using the library Tkinter and Customtkinter. The platform uses Google's Firebase to store User information. The application encrypts user passwords using a hash library and encrypts using the user's password. ChatterBox allows Users to add a friend and message them directly with real-time message updates, this is done using a "listener" method which monitors the Firebase database for updates (Changes). There is also a feature for group messaging between multiple users.


Technology used in this project:

  • Python
  • Python libraries: Customtkiner and Tkinter (Interface)
  • Python Cryptogtaphy library (User info encryption)
  • Google's Firebase (NoSQL database)
  • GitHub (version control)
